Download

Marking up a link as a download, tracking download-types with Generic GTM-package.
If a link contains the download-attribute, it will be prefixed with an arrow, and the downloaded file will have this name instead of the href-value.

Download file
 
Example
<a href="some-realy-crazy-file-name-389247823879472398.txt" download="important.txt">Download file</a>
