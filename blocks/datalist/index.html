Datalist — Work-In-Progress

To allow for quick selections in dropdowns, use an input and associate it with a datalist.

Basic use:


<label for="country-select">Select country:</label>
<input type="text" id="country-select" list="country-select-datalist" />

<datalist id="country-select-datalist">
  <option>Armenia</option>
  ...
</datalist>

Use with Awesomplete, for compability with Safari and others:


<input type="text" id="country-select" list="country-select-datalist" class="awesomplete" />
