Blocks!

Generic UI-blocks and patterns, with states controlled by radio-buttons and/or checkboxes — sometimes with a little (stand-alone) JavaScript thrown in.
Each block has it's own mixin and theme-map in SCSS.
Most blocks have built-in tracking if used with Generic GTM-container
All blocks validate on validator.w3.org
